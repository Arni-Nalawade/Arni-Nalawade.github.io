My Projects

Here are some of the engineering projects I have worked on so far

I enjoyed the process of learning new skills and implementing them as I worked on creating these projects. I am proud of how these turned out.
I endeavour to keep learning new skills and apply them to create solutions to real-life problems

Portfolio Website

I wanted to improve my web development skills and decided to create a website which showcases me and my skills. The website was made using HTML5, CSS and Javascript , in addition I also used different web development libraries and CDNs to improve functionality and performance. These incude JQuery, Google font API and Font Awesome.

BeatMaker

I wanted to expermiment using Python to create games and created this fun little minigame which uses the Pygame library to allow users to create new beat sequences from the pre-existing sounds. It also provides additional features to reload saved beats, increase tempo and number of sequences.

Hall-E

An open-ended group project which I made with my friends Nivey and Emily using C++ and Lego Mindstorms bot. Hall-E is a hallway cleaning robot capable of cleaning the very edges of the hallway where grime and dirt tends to accumulate. It is also capable of obstacle detection, auto cleaning-loops and auto-positioning.

Cloud DB-Creation

Cloud development has always interested me and hence I coded a small local cluster project which used a cloud node created in Minikube to take database creation requests from Mongoexpress and forward them to MongoDB. I learned a lot through this project especially how to code yaml files, configuring and using VMs and passing commands through Kubectl.

Flipper!

A Sumobot which I made using custom PCB made by UV radiation etching and designed using Traxxmaker software. The robot chassis was made using various woodcutting tools. It is capable of opponent detection, pit-maneuvers and edge detection. I designed this robot from the groundup and am very proud of how it turned out. It won 4th place in the local robot fighting tournament.

EduBubble

An education platform coded using HTML, CSS and DJango for Holiday-Hackathon. Me and my team wanted to create a web-application which made it easier for the teachers and the students to create different class groups and be able to make a discussion forum for class queries and projects. The application also provided a login for every user for additional security.

People Counter Circuit TinkerCAD

This is one of the many projects I coded in TinkerCAD using an Arduino Uno. Created during a time when the COVID pandemic was still ongoing the purpose of this design is to help in social distancing. It uses a light sensor to sense the number of people in the store and display instructions on whether more people can enter or if they should wait.

Measure-io

Measure-io was created as an alternate measuring device capable of measuring (with an accuracy of 4 decimal places) objects less than 1.5m in length. The program was written in C++ in Arduino and used the change in the voltage of the potentiometer derived through the moving rack and pinion to generate length calculation. The graphing simulation and calculations were done in Microsoft Excel

Catenary Bridge

As part this project, we were tasked with building a bridge which could withstand 200 kN of load using 2.5 meters of plywood and some dowels. We were able to design and build a bridge which could withstand 203 kN of load. The design simulation was done using an online 2D truss analysis tool and we created a prototype using laser cutting.
